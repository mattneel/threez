# Specs Layout

Specs are organized by epic slug:

```
docs/specs/<epic-slug>/
```

Each epic directory contains:

- `epic-brief.md`
- `core-flows.md`
- `tech-plan.md`
- `tickets.md`
- Any epic-specific supporting docs (for example `threejs-gaps.md`).

## Project Documentation

- `STATUS.md` - Current project status and architecture (source-built Dawn baseline)
- `SMOKE.md` - Cross-platform smoke testing guide
- `session-handoff.md` - Session handoff notes with timestamps

## Active Epics

- `threez` - original three.zig project spec/history
